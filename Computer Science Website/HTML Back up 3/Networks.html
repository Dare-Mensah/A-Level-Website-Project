Uniform Resource Locators(URLs) - Is the full address of an internet resource. It specifies the location of a resource the internet, including the resource name and usually the file type so the browser can request it from the website server.

Internet registers - hold records of all exisiting website names and the details of those domains that are currently avaliable to purchase.

Domain Names and the Domain Name System(DNS) - A domain name identifies the area or domain that an internet resource resides in. Thes are structured into a hireacrchy of similar of smaller domains and written as a string separated by full stops as dictated by the rules of the DNS.

Fully Qualified Domain Name(FQDN) - A fully qualified domain name is one that includes the host server name, fo rexample www.

IP Address

IP Address - is a unique address that is assigned to a network device. An IP address performs a similar functionto a home mailing address. An IP address indicates where a packet of data is to be sent or has been sent from.

Area Networks

Wide Area Networks(WANs) - i generally defined to be one that relies on third party carries or connections such as those provided by British Telecom.

Local Area Networks(LANs) - A Local Area Network consists of a number of computing devices on a single site or in a single building, connected together by cables.

Physical Bus Topology

Physical Bus Topology - A LAN can use different layouts or topologies. In a bus topolgy, all computers are connected to a single cable.

Advantages:

  • Inexpensive to install as it requires less cable than a star topolgy and doesnt require any additional hardware

Disadvantages:

  • If the main cable fails, network data can no longer be transmitted to any nodes
  • Performance degrades with a lot of traffic
  • Low security

Physical Star Toplogy

Physical Star Toplogy - A star network hasa a central node, which may be a switch or computer which acts as a router to transmit messages

Advantages:

  • Easy to add new stations without disrupting the network
  • Higher transmission speeds can give better performance than a bus network
  • If one cable fails, only one station is affected, so it is simple to isolate faults
  • No problems with 'collisions' of data since each station has its own cable to the server

Disadvantages:

  • May be costly to install because of the length of cable required
  • If the central device goes down, network data can no longer be transmitted to any of the nodes

Mesh Network Topologies

Mesh topologies are becoming more common with the widespread use of wireless technology. Each node in a mesh network has a connection to every other node, by transmitting data across any intermediate nodes.

Advantages:

  • No cabling costs<>/li>
  • New nodes are automatically incoropated into the network
  • Faster communication since data packets do not need to travel via a central switch

Packet Switching

Packet Switching - is a method of communicating packets of data across a network on which other similar communications are happening simultaneoulsy.

Circuit Switching

Circuit Switching creates a direct link between two devices for the duration of the communication.

The TCP/IP Protocal Stack

The TCP/IP protocal tack is a set of networking protocols that work as four connected layers, passing incoming and outcoming data packets up and down the layers during network communication.

Four Layers:

  • Application layer
  • Transport Layer
  • Network Layer
  • Link Layer
